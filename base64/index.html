Encode & decode

Base64 encode / decode

Paste text to encode to Base64, or paste Base64 to decode to text. Runs in the browser—nothing is sent to any server.

← Back to hub View source

How it works

Encode: Your text is converted to Base64 using the browser’s btoa() (UTF-8 handled via encodeURIComponent).

Decode: Base64 is decoded with atob(). Invalid Base64 will show an error in the output.

  • Useful for embedding small data in URLs or JSON.
  • No server round-trip—all processing is local.
