Usage

To human date: Enter a number (seconds or milliseconds since 1970-01-01 UTC). We auto-detect: if value > 1e12 we treat as ms.

To Unix: Enter an ISO date string or something Date() can parse (e.g. 2023-11-15, Nov 15, 2023).
